### What I learned

I built the layout for desktop first, then used media queries to adapt it for mobile devices:
/* Mobile -  375px  */
@media screen and (min-width: 375px) {

.container {
    padding: 0;
    margin: 0;
    border-radius: 0px;
}

/* Desktop - 1440px  */
@media screen and (min-width: 1440px) {
    
    .container {
        max-width: 736px;
        padding: 35px;
        margin: 0 auto;
     
    }
